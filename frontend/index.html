Calculation Formulas

1. Occupant Capacity (Table B1, Clause B4, PDF p.34-36)
capacity = ceil(usable_floor_area_m² / occupancy_factor)
factor from Table B1 by Use Class (e.g. office = 9 m²/person); excluded spaces (corridor/toilet/stair/lift) → capacity = 0
2. Door Width Threshold (Table B2, Clause B7.1, PDF p.43)
threshold = Table B2 lookup by occupant capacity
capacity ≤ 3 → Clause B13.4 absolute minimum 750mm; capacity > 3000 → BA case-by-case
3. Clear Width (Table B2 Note 2)
"least clear width measured between the vertical members of the door frame" — currently proxied by IfcDoor.OverallWidth (needs_human_review=true)
4. Double-leaf Check (Clause B13.4, PDF p.50-52)
estimated_leaf = measured_width / 2; must be ≥ 600mm
assumes 50/50 split; actual leaf widths need field verification
5. Status Logic
pass: width ≥ threshold AND leaf ≥ 600mm (if double-leaf)
fail: width < threshold OR leaf < 600mm
non_passage: capacity = 0/unknown OR width unknown (not an egress door)
